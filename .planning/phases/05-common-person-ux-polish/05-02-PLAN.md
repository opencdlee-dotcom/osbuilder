Implement `_parse_yaml_subset(raw: str) -> list[dict]`: a hand-rolled parser that handles the YAML structure used in dictionary.yaml. The dictionary uses YAML block sequence with `- key: value` entries. Each record starts with `- ` (list item) optionally followed by inline `key: value`. Multi-line strings use `|` block scalar. Parser algorithm:
1. Split by `^- ` (regex with MULTILINE) to get raw record blocks.
2. For each block, split by `\n` and parse `key: value` lines.
3. For block scalar `|`, collect subsequent indented lines until indent decreases.
4. Return list of dicts; first dict is the format_version metadata record.

Implement `load_dictionary(path: Path = DICTIONARY_PATH) -> list[dict]`:
```python
def load_dictionary(path: Path = DICTIONARY_PATH) -> list[dict]:
    global _DICTIONARY
    raw = path.read_text(encoding="utf-8")
    parsed = _parse_yaml_subset(raw)
    if not parsed or parsed[0].get("format_version") != "1.0":
        raise SystemExit(
            "OSBuilder: friendly-errors dictionary format-version mismatch. "
            "Expected 1.0; got " + str(parsed[0].get("format_version") if parsed else None)
        )
    _DICTIONARY = parsed[1:]
    if len(_DICTIONARY) < 30:
        raise SystemExit(
            f"OSBuilder: dictionary has {len(_DICTIONARY)} entries; expected >= 30."
        )
    return _DICTIONARY
```

Implement `_strip_tracebacks(text: str) -> str`:
```python
def _strip_tracebacks(text: str) -> str:
    lines = text.splitlines()
    out = []
    in_tb = False
    for line in lines:
        if line.startswith("Traceback (most recent"):
            in_tb = True; continue
        if in_tb:
            if line.startswith(" ") or line.startswith("\t") or 'File "' in line:
                continue
            in_tb = False
        if re.match(r"^\s+at .+\(.+:\d+\)$", line):   # Node stack frame
            continue
        if re.match(r'^\s+File ".*", line \d+,', line): # Python frame
            continue
        out.append(line)
    return "\n".join(out)
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| subprocess stderr → friendly_error.translate() | Raw subprocess output crosses this boundary; translate() must not execute, eval, or interpolate it into shell commands |
| dictionary.yaml (data file) → friendly_error.py | YAML is parsed by hand-rolled subset; format_version gate rejects tampered files |
| FriendlyMessage.copy_paste → stderr display | copy_paste is rendered as plain text only; never auto-executed |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-02-01 | Information Disclosure | _strip_tracebacks in generic_translator | mitigate | strips Python Traceback blocks and Node stack frames before including in what_broke; regex patterns tested in test stubs |
| T-05-02-02 | Tampering | dictionary.yaml format_version | mitigate | load_dictionary raises SystemExit on format_version != "1.0"; tested in test_dictionary_format_version_checked |
| T-05-02-03 | Tampering | copy_paste_command interpolation | mitigate | _build_message uses try/except KeyError on format() — malformed ctx cannot inject unexpected text; copy_paste is displayed only, never executed |
| T-05-02-04 | DoS | YAML parser on malformed file | mitigate | _parse_yaml_subset reads file as text; no recursive parsing; bounded by file size (small data file); exception → SystemExit with clear message |
| T-05-02-05 | Information Disclosure | build.log path in generic fallback | accept | what_to_do says "Check debug log at .planning/osbuilder/build.log" — path is relative and project-local; no secret leakage |
</threat_model>
